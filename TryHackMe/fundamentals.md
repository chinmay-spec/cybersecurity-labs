# TryHackMe – Fundamentals

## Objective

Learn basic networking and Linux concepts essential for cybersecurity.

---

## Networking Basics

A network is a group of connected devices that communicate with each other.

* **IP Address** → Identifies a device on a network
* **DNS (Domain Name System)** → Converts domain names into IP addresses
* **HTTP / HTTPS** → Protocols used for web communication

---

## Linux Basics

Linux is widely used in cybersecurity for penetration testing and system administration.

### Commands Practiced

* `whoami` → Shows the current user
* `pwd` → Displays current directory
* `ls` → Lists files and directories
* `cd` → Changes directory

---

## Key Takeaways

* Networking is the foundation of cybersecurity
* Linux is the primary environment for security tools
* Understanding these basics is essential before advanced topics
