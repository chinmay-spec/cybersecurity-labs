# Linux File Management

## Objective
Understand file creation, viewing, copying, moving, and deletion in Linux.

---

## Commands Practiced

- touch
- ls
- ls -l
- cat
- cp
- mv
- rm
- mkdir
- rmdir
- tree

---

## Key Concepts

### File Creation
`touch filename`
Creates an empty file.

### Viewing Files
`cat filename`
Displays file contents.

### Copying Files
`cp source destination`
Used for duplicating files.

### Moving/Renaming
`mv oldname newname`
Moves or renames files.

### Deleting Files
`rm filename`
Removes file permanently.

---

## Security Relevance

Understanding file operations is critical in:
- Log analysis
- Malware investigation
- File integrity monitoring
- Evidence collection
